# Imagem Open Graph

Esta pasta deve conter uma imagem `og-image.png` com as seguintes especificações:

- **Tamanho**: 1200x630 pixels
- **Formato**: PNG ou JPG otimizado
- **Conteúdo sugerido**: 
  - Logo do Rachid
  - Texto: "Rachid - Divisão de despesas em grupo"
  - Cores consistentes com o branding (gradiente azul/verde)
  - Design limpo e profissional

**Ferramentas recomendadas para criar:**
- Canva (template: Open Graph Image)
- Figma
- Photoshop
- GIMP

**Após criar, salvar como:** `landing/public/og-image.png`
