# Landing Page - Rachid

Landing page estática gerada com Astro para orachid.com.br

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

A landing está configurada para deploy na Vercel. O arquivo `vercel.json` já está configurado.

**Importante:** Antes do deploy, certifique-se de criar a imagem OG em `public/og-image.png` (1200x630px).

## Estrutura

- `src/pages/` - Páginas Astro
- `src/layouts/` - Layouts base
- `src/components/` - Componentes reutilizáveis
- `src/styles/` - Estilos CSS
- `public/` - Arquivos estáticos (robots.txt, sitemap.xml, imagens)
